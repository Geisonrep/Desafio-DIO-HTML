Estrutura Básica do HTML

Voltar

HTML é uma linguagem de marcação de hipertexto (HiperText Markup Language) utilizada na construção de páginas na Web. O HTML permite inserir conteúdo e estabelecer uma estutura básica de um website. Atualmente utilizamos a versão 5 do HTML.

A estutura básica de um documento HTML, deve conter uma declaração < !DOCTYPE html>, segue abaixo a estrutura com as definições:

  1. html

    É iniciada após do DOCTYPE e fechada ao final do documento.

  2. head

    É um elemento que receberá as informações básicas da página, como o título, metadados, links de elementos exeternos e etc. São dados implícitos e dentro dela, necessária presença do title. Podemos configurar também qual será a codificação que iremos utilizar no documento, por meio do charset UTF-8.

  3. title

    É onde escrevemos o título do website, fica escrito na aba do navegador.

  4. body

    É o corpo do nosso documento, todos os elementos que serão renderizados no navegador, será através do body. Nele colocamos todo o texto, imagens, vídeos, etc. Dentro desse elemento, são declarados outras tags, como cabeçalhos, listas, rodapé, etc.

Falando sobre Tags

Voltar

A tag é representada por sinal de menor e maior " < > ". São um conjunto de caracteres que formam um elemento. A maioria das tags HTML, são compostas por uma estrutura de abertura e fechamento, como < tag> e < /tag>.

Existem também outras tags que são de estrutura única, ou seja, não precisa de fechamento. Mas por convenção, representamos de uma forma parecida com a tag de fechamento: < tag />.

Tipos de Tags

  • < h1> Cabeçalho < /h1>: Tag para definir um título. Essas tags podem variar de 1 a 6, < h1>, < h2>, < h3>, < h4>, < h5> e < h6>. Abaixo temos uma sequencia dos tamanhos de h, começando pelo < h1> e finalizando com < h6>:

    Exemplo de h1;

    Exemplo de h2;

    Exemplo de h3;

    Exemplo de h4;

    Exemplo de h5;

    Exemplo de h6.

  • < p> Parágrafo < /p>: Tag para definir um parágrafo;

  • < a> Link < /a>: Tag de link. Dentro dessa tag, é necessário ter uma atribuição "href=", esse assunto será abordado no próximo tópico;

  • < ol> Lista ordenada < /ol>: Tag que define uma lista ordenada, ou seja, tem numeração;

  • < ul> Listas desordenadas < /ul>: Tag de listas desordenadas, elas aparecem com um ponto;

  • < h1> Cabeçalho < /h1>: Tag de cabeçalho;

  • < h1> Cabeçalho < /h1>: Tag de cabeçalho;

Atritubo das Tags

Voltar

Os atributos HTML são palavras (especiais) utilizadas dentro das tags de abertura. Através desses atributos, conseguimos aplicar uma classe, id, lang, style e etc. É possível ver na documentação de HTML.

Segue abaixo alguns exemplos:

  • < a hre="">: Dentro do sinal de aspas "", é o local onde colocamos o endereço do website;

  • < img src="">: Dentro do sinal de aspas "", colocamos o local da imagem;

Efeitos e Formatações em Textos

Voltar

Alguns efeitos não são utilizados usualmente, porém, em alguns casos especiais, se faz necessário colocar alguma decoração ou efeito no texto. Essas tags servem para estilizar o seu conteúdo.

Segue abaixo alguns exemplos:

  • A tag < i> dá um efeito em itálico < /i>;

  • A tag < strong> dá um efeito em negrito < /strong>;

  • A tag < mark> dá um efeito de marcação < /mark>;

  • A tag < del> dá um efeito de rabisco < /del>;

  • A tag < sup> coloca o texto ou palavra na parte superior < /sup>;

  • A tag < sub> coloca o texto ou palavra na parte inferior < /sub>;

  • A tag de abreviação, < abbr> RJ < /abbr> falcilitr o mecanismo de busca e sistema de tradução, podemos colocar um atributo title="Rio de Janeiro", para que seja relevante nos navegadores. Ela dá o significado semântico a palavra do texto HTML.

Listas

Voltar

São elementoss de textos de grande importância na criação de websites, esse modelo de organização, facilita o entendimento e cria uma boa experiência para o usário. Em alguns casos, se torna praticamente obrigatório, como no caso de uma receita culinária.

Iremos abordar sobre itens, lista ordenada e lista desordenada.

Tag li

"li" significa list item em inglês. Essa tag < li> é usada para cada item da lista, ela possui abertura e fechamento, como: < li> e < /li>.

Listas ordenadas

Essa tag é "ol" e significa ordered list em inglês. No exemplo abaixo, está sendo demonstrado a estrutura de uma lista ordenada com os itens:

< ol>

< li> elemento < /li>

< /ol>

  1. Elemento I
  2. Elemento II
  3. Elemento III
  4. Elemento IV

Listas desordenadas

Essa tag é "ul" e significa unordered list em inglês. No exemplo abaixo, está sendo demonstrado a estrutura de uma lista desordenada com os itens:

< ul>

< li> elemento < /li>

< /ul>

  • Elemento I
  • Elemento II
  • Elemento III
  • Elemento IV

Links

Voltar

A tag link HTML é uma função que permite inserir hiperlinks em vários alguns elementos, como imagens e textos. O link precisa de um atributo href="", pois é desta maneira que apontamos uma URL (endereço).

Ela é escrita da seguinte forma:

< a href="https://web.dio.me/home"> DIO < /a>

Atributo href

O atributo href tem o intuito de redirecionar o usuário a outro recurso ou documento. Existes 3 tipos de links utilizados para esse redirecionamento. Estes tipos são:

  • Interno:Redireciona para um elemento dentro da mesma página;

    < a href="#contato"> Contato < /a>

  • Local:Redireciona para um elemento dentro da mesma página;

    < a href="../pg/pagina2.html"> Pagina 2 < /a>

  • Global:Redireciona para um elemento dentro da mesma página;

    < a href="http://www.google.com/"> Google < /a>
